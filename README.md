# RectYX2D

RectYX2D is an objects animations calculator engine, designed to create 2D animations, like calculations to create video games or web animations. RectYX2D calculate coordinates, objects moves and collisions beetween objects in 2D world.

RectYX2D is not a rendering engine.

Latest version of RectYX2D is 1.0.1.

### Requirements

OpenJDK.

RectYX2D tested OK with Eclipse Temurin (OpenJDK build) 11 and 17 versions, and should also run with any later versions.

### Free to use and to modify

RectYX2D is free to use and to modify, including for commercial usage, according to the terms of Apache License, Version 2.0.

### External library usage with separated code

RectYX2D is designed to perform fondamental, repetitive and non specifics calculations.
The application that import RectYX2D perform mostly the specifics and the events calculations.
The application that import RectYX2D also import display library (like libGDX) aside of RectYX2D.

### Internal code usage

RectYX2D source code also can be modified internaly as desired to implement application logic and display library.
